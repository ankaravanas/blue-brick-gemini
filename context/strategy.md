# Strategy — Blue Brick Website Project

## Current Focus

Build a premium, visually striking website for Blue Brick that captures the venue's hybrid all-day concept and establishes its digital presence ahead of / during launch (March 2026).

## Project Goals

1. **Create a best-in-class hospitality website** — Modern, immersive, on-brand
2. **Communicate the "Sips, Bites, Sounds" concept** — Make the hybrid nature clear and compelling
3. **Capture the "Break the Brick" attitude** — Bold, anti-stereotype, intentional design
4. **Drive real-world action** — Visits, reservations (if applicable), social follows

## Technical Approach

### Stack (following DR8 reference)
- **Vite + React + TypeScript** — Fast, modern, type-safe
- **Tailwind CSS + shadcn/ui** — Utility-first styling with quality components
- **Framer Motion / GSAP** — Smooth, premium animations
- **Single-page sections** or minimal routing

### Reference
- **DR8 Athens** — Sister venue, same street, similar premium nightlife/hospitality positioning
- Repo cloned to `reference/dr8-athens-vibe/`
- **Do NOT copy** — Use as inspiration for structure, animation quality, and visual bar
- Blue Brick needs its own distinct identity (blue palette vs red, all-day vs nightclub, hybrid concept)

## Key Sections (Likely)

Based on the concept and DR8 reference:
- Hero (logo, tagline, vibe)
- Concept / About (Sips, Bites, Sounds pillars)
- Menu preview or link
- Gallery
- Location / Contact
- Social links / Footer

## What Success Looks Like

- A website that matches or exceeds DR8's quality
- Distinct Blue Brick identity — not a color swap of DR8
- Smooth performance on mobile and desktop
- Client happy with the result
- Site ready for launch timing
